📝 How to Draw (ChemDraw Style):

  • Add Atoms: Select atom type (C, O, N, etc.) → Click to place (snaps to grid)
  • Smart Bonds: Select bond type → Click first atom → Click second atom OR empty space (auto-creates new atom with angle/length snap)
  • Quick Structures: Benzene, cyclopentane, cyclohexane, cycloheptane, or carbon chain templates
  • Grid Snap: Atoms snap to 20px grid for perfect alignment (toggle off if needed)
  • Angle Snap: Bonds snap to 30° increments for standard bond angles (0°, 30°, 60°, 90°, 120°, 150°)
  • Fixed Length: Bonds default to 50px length for consistent structure (toggle off for free drawing)
  • Delete: Right-click atoms or bonds to remove them
  • Change Atoms: Click existing atom with different element selected to change it
